# Simple Shop (React + Node + MongoDB)

This project contains a minimal ready-to-run shop:
 - server/  - Node.js + Express API
 - client/  - Vite + React frontend

Quick start:
1. Make sure Node.js and npm are installed.
2. Start MongoDB locally or use Atlas. Configure server/.env if using remote.
3. Open two terminals:

Server:
cd server
npm install
npm run dev

Client:
cd client
npm install
npm run dev

Server runs on http://localhost:5000
Client (Vite) usually on http://localhost:5173
